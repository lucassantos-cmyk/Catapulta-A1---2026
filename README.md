Catapulta - Grupo A1


Funcionalidades incluídas em catapulta_wifi.ino (versão 1.0):
- Atualização de código via wi-fi com atualizações OTA (Over-the-Air)
- Comunicação bluetooth com aplicativo para controle das funcionalidades da catapulta
- Controle dos componentes eletrônicos conectados à catapulta

Funcionalidades incluídas em catapulta.apk
- Comunicação bluetooth com módulo esp
- Controle do servo motor que vai travar ou destravar a catapulta para o lançamento
- Controle do cilindro que vai regular a tensão dos elásticos a partir do número de voltas que se espera para esse motor
